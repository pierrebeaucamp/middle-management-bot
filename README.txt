Middle Management Bot

This is a work-in-progress GitHub Probot application to automate some common
busywork. I'm primarily writing this for personal use, but ideas and
contributions are welcome.

This project uses TravisCI for continuous integration. The latests builds can be
found at https://travis-ci.com/pierrebeaucamp/middle-management-bot.
